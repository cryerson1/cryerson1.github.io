About the Nike ZoomX Invincible Run Flyknit

The Nike ZoomX Invincible Run is changing the game among highly cushioned daily trainers. With an innovative concept which optimally blends cushioning, responsiveness, and support, the Invincible delivers the ultimate daily training package with a unique underfoot experience.This shoe is loaded with a very soft full-length ZoomX midsole that has the most energy return out of any Nike foam. The knit upper, rocker shape, and wider base allow this shoe to be an amazing option for any runner trying to optimize their training.
